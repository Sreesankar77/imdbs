# IMDBS - Movie Review App (Minimal)

Quick start:

1. Create MySQL database named `imdbs_db` and ensure user `sankar` with password `sankar_goat` has access.

2. Install Python deps:

```bash
python3 -m pip install -r requirements.txt
```

3. Run the app:

```bash
export FLASK_APP=app.py
flask run
```

4. Default admin credentials: username `sankar`, password `12345678`.

Notes:
- Uploads are stored in `static/uploads`.
- The UI uses a dark background and red buttons as requested.
- This is a minimal scaffolding. You can extend validation, search, and image handling as needed.
